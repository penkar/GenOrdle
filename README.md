# GenOrdle - The React Wordle Clone

This is a not for profit clone of the NYT [Wordle](https://www.nytimes.com/games/wordle/index.html) game.


## How to start the game:

### Run `yarn start`
Without modification the application can be viewed on [http://localhost:3000](http://localhost:3000).

## How to create a deployable version of the game:

### Run `yarn build`
